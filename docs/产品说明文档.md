> 版本：v1.0 | 更新日期：2026-06-07 | 平台：微信小程序

## 一、产品概述

### 1.1 产品定位

**灵光**是一款面向个人创作者的**灵感捕捉与管理工具**。它帮助用户在灵感闪现的第一时间快速记录，并通过标签、项目分类、吸收标记等机制进行系统化管理，最终促进灵感的消化与转化。

### 1.2 目标用户

- 内容创作者（写作者、设计师、视频博主）
- 知识工作者（产品经理、研究员、学生）
- 任何需要快速记录碎片化想法的人

### 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| **极速捕捉** | 打开即记，支持文字、图片、语音多种输入 |
| **结构化沉淀** | 标签自动提取、项目分类收纳、吸收状态追踪 |
| **精美分享** | Apple Card 风格海报，一键分享灵感卡到微信 |
| **数据安全** | 本地优先 + 云端同步，基于版本号的冲突解决 |

## 三、页面结构

### 3.1 页面路由

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | `pages/index/index` | 灵感流 + 项目列表（核心页面） |
| 详情 | `pages/detail/detail` | 灵感详情、补充、海报 |
| 搜索 | `pages/search/search` | 关键词搜索、标签筛选 |
| 设置 | `pages/settings/settings` | 账号、模式、资料 |

### 3.2 首页 — 双模式设计

首页根据**项目模式**呈现两种不同视图：

**单项目模式（默认）**：
- 顶部：品牌 Logo + 搜索/设置入口
- 中上：文本输入区 + 捕捉按钮
- 中部：灵感流列表（按日期分组）
- 显示吸收/未吸收统计

**多项目模式**：
- 顶部：三栏布局 `[我的项目(N)] [X未吸收/N] [排序][+]`
- 中上：项目网格 + 未分类入口
- 点击项目 → 进入该项目灵感流（子页面视图）
- 点击未分类 → 展示无归属灵感

### 3.3 页面导航关系

```
                    ┌──────────┐
                    │   首页   │
                    └────┬─────┘
           ┌─────────────┼─────────────┐
           ▼             ▼             ▼
     ┌──────────┐  ┌──────────┐  ┌──────────┐
     │   详情   │  │   搜索   │  │   设置   │
     └──────────┘  └──────────┘  └──────────┘
```

- 首页 → 详情：点击灵感卡片
- 首页 → 搜索：点击搜索图标
- 首页 → 设置：点击设置图标
- 详情 → 首页：返回按钮
- 搜索 → 详情：点击搜索结果
- 设置 → 首页：切换模式后自动重载

## 五、功能流程

### 5.1 创建灵感

```
打开小程序
    │
    ├── 有草稿？→ 恢复输入内容
    │
    ▼
输入文本（自动提取 #标签）
    │
    ├── 可选：添加图片（最多9张）
    │
    ▼
点击"捕捉"
    │
    ├── 生成 localId
    ├── 提取标签（正则匹配 /#[\u4e00-\u9fa5\w]+/g）
    ├── 获取地理位置（需授权）
    ├── 保存到本地 storage
    ├── 加入同步队列
    ├── 清空输入框 + 草稿
    └── 刷新列表
```

### 5.2 灵感管理操作

| 操作 | 触发方式 | 效果 |
|------|---------|------|
| 查看详情 | 点击卡片 | 跳转详情页 |
| 标记吸收 | 右滑卡片 / 详情页切换 | 标记已吸收（绿色勾） |
| 删除 | 左滑卡片 / 长按菜单 / 详情页 | 软删除，不丢失数据 |
| 复制内容 | 长按菜单 → 复制 | 复制正文+补充到剪贴板 |
| 分享灵感卡 | 长按菜单 → 分享 / 详情页 | 跳转详情并打开海报 |
| 批量标记吸收 | 管理 → 勾选 → 标记吸收 | 批量设为已吸收 |
| 批量删除 | 管理 → 勾选 → 批量删除 | 确认后批量软删除 |
| 转为项目 | 管理 → 勾选 → 转为项目 | 创建新项目并归入 |

### 5.3 项目管理操作

| 操作 | 触发方式 | 效果 |
|------|---------|------|
| 创建项目 | 点击 + | 输入名称创建 |
| 进入项目 | 点击项目卡片 | 进入该项目灵感流 |
| 重命名 | 长按卡片 → 重命名 | 修改项目名称 |
| 删除项目 | 长按卡片 → 删除 | 确认后删除，灵感变未分类 |
| 拖拽排序 | 点击排序 → 拖拽 → 完成 | 持久化新顺序 |
| 切回首页 | 项目页点返回 | 回到项目列表 |

### 5.4 搜索流程

```
进入搜索页 → 自动聚焦输入框
              │
    ┌─────────┴─────────┐
    ▼                   ▼
输入关键词            点击标签
    │                   │
    └─────────┬─────────┘
              ▼
    实时搜索（内容 + 补充 + 标签匹配）
              │
              ▼
    展示结果（含项目标签 + 吸收状态）
              │
              ▼
    点击结果 → 跳转详情页
```

### 5.5 分享海报生成

```
详情页 → 点击"分享灵感卡"
    │
    ├── 并行下载图片
    ├── 并行获取小程序码
    │
    ▼
Canvas 2D 绘制
    ├── 顶部：品牌紫色渐变背景
    ├── 标题区：slogan + 日期
    ├── 正文区：灵感内容（自动折行）
    ├── 图片区：网格布局（3列，圆角裁剪）
    ├── 补充区：灰色卡片样式
    ├── 位置区：📍 城市·区域
    └── 底部：小程序码 + 引导文字
    │
    ▼
destination-in 圆角裁剪（28px）
    │
    ▼
展示海报弹层
    ├── 保存到相册（wx.saveImageToPhotosAlbum）
    └── 发送给朋友（wx.showShareImageMenu）
```

### 5.6 数据同步

```
触发时机：
  - 登录后
  - 创建/修改/删除灵感后
  - 从设置页手动同步

上传同步 (syncUpload)：
  syncQueue → 构建灵感数组 → 云函数 syncUpload
  → 接收 localId→cloudId 映射
  → 更新本地 synced=true, cloudId=xxx
  → 清空同步队列

下载同步 (syncDownload)：
  lastSyncAt → 云函数 syncDownload
  → 遍历云端灵感
  → 本地不存在 → 插入
  → 云端 version > 本地 version → 覆盖
  → 更新 lastSyncAt
```

## 七、技术说明

### 7.1 平台依赖

- 微信小程序基础库
- 微信云开发（云函数 + 云存储）
- 腾讯地图 API（地理位置）
- Canvas 2D（海报绘制）

### 7.2 数据策略

- **本地优先**：所有读写先操作本地 storage，保证离线可用
- **同步队列**：未同步操作暂存 syncQueue，联网后逐条上传
- **版本号冲突解决**：version 严格大于才覆盖，防止本地修改丢失
- **软删除**：删除操作仅标记 isDeleted=true，不物理删除

### 7.3 性能优化

- 灵感列表按日期分组，避免一次渲染过多条目
- 图片使用临时路径，节省存储空间
- 海报 Canvas 复用，避免重复创建
- 搜索实时防抖

## 八、版本兼容

### 8.1 配色迁移

- 旧版本配色 → v2 升级：`migrateProjectColors()` 自动重新分配配色
- 首次启动时执行，标记 `__project_color_migrated_v2__`，只执行一次

### 8.2 数据升级

- 灵感字段向后兼容，新增字段设默认值
- 项目模式切换不影响已有数据
- 多→单模式下，灵感保持原有 projectId，默认取第一个项目
